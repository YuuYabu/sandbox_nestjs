# Sandbox NestJS

## 前提

- WSL2(Windows)
- Docker Desktop
- VS Code / Cursor
  - `Dev Containers`拡張機能

## 開発コンテナー起動

1. `Dev Containers` 拡張機能をインストール
2. Clone後左下歯車マークの下の青いタイルをクリック
3. `コンテナーを再度開く` を選択
4. ロードが終わるまで待つ

## NestJSアプリ起動

以下二通りの方法で起動

1. ターミナルで `npm run start:dev` を実行
2. VS Codeのエクスプローラーの下にある `NPMスクリプト` タブから `start:dev` を実行
